## 1. DDL (Data Definition Language)
Defining the structure.
*   `CREATE TABLE`: New table.
*   `ALTER TABLE`: Add a column.
*   `DROP TABLE`: Delete table structure.
*   `TRUNCATE`: Delete ALL data but keep structure.

## 2. DML (Data Manipulation Language)
Managing the data.
*   `INSERT INTO`: Add rows.
*   `UPDATE`: Edit rows.
*   `DELETE`: Remove rows.

## 3. DQL (Data Query Language)
*   `SELECT`: Read data.

## 4. TCL (Transaction Control)
*   `COMMIT`: Save changes permanently.
*   `ROLLBACK`: Undo changes (Ctrl+Z).

## 5. Joins Visualized
*   **INNER**: Venn diagram intersection (Only Matches).
*   **LEFT**: All from Left Table + Matches from Right.
*   **FULL**: Everything from both tables.
